## 功能介绍 
    
大学班车预约小程序：学校全面实施交通班车预约乘坐  使用班车预约系统  提早规划出行  大幅减少排队等待的时间  三校区往返更加便捷有效

- 预约管理：开始/截止时间/人数均可灵活设置，可以自定义客户预约填写的数据项
- 预约凭证：支持线下到场后校验签到/核销/二维码自助签到等多种方式
- 详尽的预约数据：支持预约名单数据导出Excel，打印
